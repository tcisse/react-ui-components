# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2025-01-XX

### Added
- **PricingTable component** - A versatile pricing comparison table component with multiple variants
  - `table` variant: Classic comparison table with feature rows
  - `cards` variant: Modern card-based layout with side-by-side plans
  - `minimal` variant: Clean and minimal design for a subtle look
  - `featured` variant: Prominent featured plan with other plans below
- Support for multiple pricing plans with customizable features
- Feature comparison with checkmarks, X icons, and custom values
- Plan highlighting with badges and popular indicators
- Responsive design (tablet/mobile optimized)
- TypeScript support with full type definitions

## [0.1.2] - 2025-02-06

- useClickOutside hook for detecting clicks outside elements
